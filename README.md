**Mealy-Zone** is a modern food recipe web app built with **React** and **Tailwind CSS**.  
It allows users to browse meals by category, view detailed recipes, and save their favorite dishes for quick access.  

Live Demo: [https://mealy-zone.netlify.app](https://mealy-zone.netlify.app)

---

## Features

- Browse meals by **categories**  
- View **detailed recipe information** (ingredients, instructions, images)  
- Mark recipes as **favorites** for quick access  
- Pagination for large meal lists  
- Fully responsive UI powered by **Tailwind CSS**  
- Smooth navigation with **React Router**

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Rayyan-Arif/Mealy-Zone.git
```

2. Navigate into the project folder:

  cd Mealy-Zone


3. Install dependencies:

  npm install


4. Run the development server:

  npm run dev


5. Open your browser at:

  http://localhost:5173
